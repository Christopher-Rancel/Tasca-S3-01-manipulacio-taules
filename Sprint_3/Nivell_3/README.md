# Nivell_3

### Exercici_1
Adapta el model de la base de dades per tal que coincideixi amb el diagrama EER proporcionat.

- Crea la taula `user` a partir dels arxius `estructura_dades_user` i `dades_introduir_user`
- Insereix les dades dels usuaris
- Assegura la compatibilitat de tipus de dades entre `user` i `transaction`
- Afegeix els registres necessaris per mantenir la integritat referencial
- Crea la relació entre `transaction` i `user`
- Modifica la taula `credit_card` afegint la columna `fecha_actual`
- Comprova l’estructura final de la base de dades i genera el diagrama EER

### Exercici_2
Crea una vista anomenada `InformeTecnico` que contingui la següent informació:

- ID de la transacció  
- Nom de l’usuari/ària  
- Cognom de l’usuari/ària  
- IBAN de la targeta de crèdit utilitzada  
- Nom de la companyia associada a la transacció  

Mostra els resultats de la vista ordenats de forma descendent segons l’ID de la transacció.
